# Gagan Dhawan Next.js Website

## Stack
- Next.js App Router
- Bootstrap via CDN
- Normal CSS
- Swiper for ventures and blogs
- Iconify for social icons
- Nodemailer for contact form
- Inter font via next/font/google

## Run locally
```bash
npm install
npm run dev
```

## Contact form setup
1. Copy `.env.example` to `.env.local`
2. Add SMTP credentials
3. Start the app

### Easy free SMTP option
Use Brevo SMTP free tier and place those credentials in `.env.local`.

## Structure
- `app/` app router pages and API routes
- `components/` page sections
- `data/` placeholder content
- `public/images/` placeholder assets
- `styles/home.css` all custom styling
